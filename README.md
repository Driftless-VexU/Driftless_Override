# Driftless Override

PLATT1 - Driftless  
Software for the 2026 - 2027 VexU season, Override

## How to run
This project requires the following libraries:  
PROS 4.2.2  
Eigen 5.0.0  
  
Once installed, the project can be built and uploaded to a vex brain using the PROS VSCode extension or the PROS cli.
