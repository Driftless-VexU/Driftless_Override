# Driftless Override

PLATT1 - Driftless  
Software for the 2026 - 2027 VexU season, Override

## How to run
This project requires the following libraries:  
PROS 4.2.2  
Eigen 5.0.0  
  
Once installed, the project can be built and uploaded to a vex brain using the PROS VSCode extension or the PROS cli.  

## Contribution Guidelines
To contribute, create an issue using the following format:  
```{Bug/Refactor/Feat}: {iteration} {issue name}```  
  
Keep issue names concise. If more detail is needed, give a more in-depth description within the description section, not the title.  
  
If you wish to work on an issue, branch from Dev, using the issue name as the branch name, i.e.  
```#5-Bug-V2-Port-Errors```  
to ensure proper linking.  

While working in a branch, begin commit messages with the related issue id.  
When you are done with an issue, create a pull request titled:  
```Closed issue {issue id}```
